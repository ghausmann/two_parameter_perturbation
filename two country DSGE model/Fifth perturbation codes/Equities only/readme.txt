The current codes in this folder are:

prepare_model_dsge2.m: script, performs the pre-processing of the two-country DSGE model with equities only.
eval_sss_dsge2.m: evaluation function to implement the SSS algorithm, with symmetric countries.
compute_sss_dsge2.m: function to approximate the SSS of bonds and equities, with symmetric countries.
solution_dsge2_compstats.m: script to perform comparative statics (Figure 1).

INSTRUCTIONS: 
- Add the folders "Perturbation" (folder and subfolders) and "common functions" to the Matlab path.
- Run the script prepare_model_dsge2.m once.
