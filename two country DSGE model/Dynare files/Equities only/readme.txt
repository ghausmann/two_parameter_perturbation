The current files in this folder are:

prepare_model_dsge2.mod: mod file of the two-country DSGE model with equities only.
eval_sss_dsge2_dyn.m: function to implement the algorithm described in Section 2.4 of the paper.
compute_sss_dsge2_dyn.m: function to approximate the SSS of equities.

pre_processing_dsge2.m: script to do the pre-processing of the two-country DSGE model.

solution_dsge2_compstats_dyn.m: script to perform comparative statics.
